# Automata Editor

A second attempt at this...

A web tool that can edit and visualize various Automata.

It has some additional features to support its use in the Wits University FLA course.

## Planned Support

- [ ] Finite Automata
- [ ] Moore / Mealy Machines
- [x] Decision Turing Machines 
    - [x] Two Way Tape
    - [ ] One Way Tape
    - [x] Multi-step / Stay Instructions
    - [ ] Deterministic
    - [ ] Non-Deterministic
    - [ ] Verifiers
    - [ ] Multi-Tape
- [ ] Computational Turing Machines

## Build and Run

1. Svelte

```
npm install
npm run dev
```
